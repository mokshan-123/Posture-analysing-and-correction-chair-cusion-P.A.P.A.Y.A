# Posture-analysing-and-correction-chair-cusion-P.A.P.A.Y.A
PulseCode (P.A.P.A.Y.A.) is a smart posture cushion designed to monitor sitting posture in real time using pressure and distance sensors. It analyzes seating patterns through an embedded microcontroller and provides gentle vibration feedback to help users maintain healthy posture habits in a non-intrusive way.
